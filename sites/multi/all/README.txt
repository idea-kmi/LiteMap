
In this folder you can make changes that will be applied to all multi domains

1.	If you want to create a theme that all multi domain sites use
	place it in the 'theme' folder in this 'all' folder, 
	(see the README.txt file in 'theme' folder for more info).
2.	If you want to override interface text elements for all the domains to display
	place the overrides in the 'language' folder in this 'all' folder,
	(see the README.txt file in 'language' folder for more info).
3.	If you want to make code changes for all domains to share
	place them in the 'custom' folder in this 'all' folder,
	(see the README.txt file in 'custom' folder for more info).
